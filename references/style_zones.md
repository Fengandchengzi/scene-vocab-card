# Style Zone 速查表

直接复制对应风格的 prompt 文本，替换 scene_image_prompt 中的【风格区】。

---

## 通用风格原则（适用于所有风格）

无论选择哪种 style_zone，以下原则必须遵守：

1. **文字可读性优先**：标签卡使用干净的白色/米色底（off-white / cream），留足间距，字体清晰锐利，画面不拥挤。大词汇量时分区布局，避免标签重叠
2. **课堂适用**：warm, polished, child-friendly, readable, classroom-ready。色彩明快但不刺眼，适合打印和投屏
3. **IP 安全**：不使用受保护的 IP 角色名、品牌名或 Logo。如用户要求特定 IP 风格（如"疯狂动物城"），转为原创等价描述：
   - "疯狂动物城人物" → "原创拟人动物角色，温暖 3D 动画风格"
   - 禁止出现 Disney / Pixar / Zootopia 等品牌字样
   - 在方案中告知用户已做 IP 安全转换
4. **避免项**：no photorealistic rendering, no Chinese characters in labels (Chinese allowed only in title banner), no real brand logos, no frame borders, no watermark, no duplicate labels, no misspellings, no random extra words, no irrelevant decorative text, no grid or collage layout

## 1. default — 现代教育插画

```
modern educational illustration style, clean vector-based art with soft hand-drawn texture, gentle rounded shapes, warm and friendly color palette (muted terracotta, sage green, cream white, warm ochre), soft even lighting (no harsh shadows), subtle grain texture overlay, 9:16 portrait ratio, no photorealistic elements, no dark or dramatic lighting, no cinematic atmosphere
```

## 2. watercolor — 柔和水彩

```
soft watercolor illustration style, gentle transparent washes with visible paper texture, delicate ink outlines, pastel and muted tones (dusty rose, soft teal, warm cream, light lavender), natural color bleeding at edges, white paper showing through in highlights, hand-painted feel with slight imperfections, 9:16 portrait ratio
```

---

## 3. hand_painted_anime — 手绘动画风

```
Japanese hand-painted animation art style, rich atmospheric backgrounds with meticulous detail, soft cel-shading with warm color grading, lush natural elements (grass, clouds, sunlight), gentle and nostalgic mood, painterly sky with layered clouds, characters with simple expressive features, 9:16 portrait ratio
```

---

## 4. line_drawing — 黑白线描

```
black and white line drawing illustration, clean precise ink lines with varying thickness, minimal crosshatching for shading, generous white space, no color fill (pure linework), architectural precision for objects, loose gestural lines for figures, subtle stippling for texture, 9:16 portrait ratio
```

---

## 5. pixel_art — 像素画

```
16-bit pixel art style, limited color palette (max 32 colors), crisp pixel edges with no anti-aliasing, retro game aesthetic, isometric or side-view perspective, chunky readable shapes, dithering for gradients, nostalgic warm tones, 9:16 portrait ratio
```

## 6. chinese_ink — 中国水墨

```
traditional Chinese ink wash painting style (shui-mo), xuan paper texture with natural ink diffusion, varying ink density from deep black to pale grey washes, minimal color accents (vermillion seal, pale mineral pigments), generous negative space (liu-bai), calligraphic brushwork for lines, contemplative and elegant atmosphere, 9:16 portrait ratio
```
